UI not bundled

This remote-signer binary was built with make build-cli (Go-only, no embedded operator web UI).

To get the real UI:

  • Default / production: make build (or make build-embed) — runs vite then re-builds with -tags embed_web.
  • Local dev: cd web && pnpm dev and open http://localhost:5173 — Vite's dev server proxies /api/* to this daemon.

The HTTP API at /api/v1/*, /health, and /metrics works the same in either build.
